# 多数据库支持

多数据库支持可以帮助开发者高效的从一种数据库迁移到另一个数据库，减少了数据库工程师维护的工作，为企业和开发者节约成本

## 当前支持的数据库目录
```
SQLServer
MySql
Sqlite
PostgreSql
Oracle
Dm（达梦）
```
## 默认数据库配置

NCF 第一次拉取后默认配置为 Sqlite，[配置方法](/start/database/setting.html)
