# How to Call Between Modules

The inter-module calls mentioned here refer to the calls between Xncf Modules. Below, we will create two modules `Senparc.Xncf.Teacher` and `Senparc.Xncf.Student`, and then use the Teacher module to call the Student module.

## Create 2 Modules

Follow the [Create a Module Chapter](/start/xncf-develop/create-xncf.html) to create the two modules `Senparc.Xncf.Teacher` and `Senparc.Xncf.Student`.
