# Senparc.AI

## AiPlatform介绍

当前可用的 `AiPlatform` 属性的参考值

| AiPlatform类型 | 说明                      |
| -------------- | ------------------------- |
| UnSet          | 不设置                    |
| None           | 设置为空                  |
| Other          | 其他                      |
| NeuCharAI      | 盛派 `NeuChar` 平台 `AI`  |
| OpenAI         | OpenAI                    |
| AzureOpenAI    | AzureOpenAI（简称：AOAI） |
| HuggingFace    | HuggingFace               |
| FastAPI        | FastAPI                   |
| Ollama         | Ollama                    |

## AzureOpenAI 配置

进入 `https://portal.azure.com/#home`
