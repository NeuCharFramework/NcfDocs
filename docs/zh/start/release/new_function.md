## 一键还原(开启)扩展模块的菜单项

对于开发来说难免还需要再次进行安装其他的模块，进行调试，跟踪，那么如何还原呢？

重新访问：http://你的域名/Admin/XncfModule/Index

![Image text](./images/new_function/show_open_extend_menu_button.png)

点击开启即可，是不是 So Easy.
